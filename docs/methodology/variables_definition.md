# Definición de Variables del Modelo Empírico

## Propósito del documento

Este documento define la **arquitectura conceptual de las variables** del modelo empírico del proyecto.
No constituye un diccionario de datos, sino un **mapa teórico-empírico** que articula:

- constructos teóricos
- conceptos económicos
- mecanismos causales
- dimensiones analíticas
- estructuras empíricas

La definición operativa final de cada variable se realizará una vez construido el panel de datos.

---

## Principio metodológico

Las variables no se definen a partir de la disponibilidad de datos, sino a partir del **modelo teórico** y del **marco conceptual** de la investigación.

Los datos serán seleccionados y armonizados en función de estas definiciones conceptuales.

---

## Estructura del modelo empírico

El modelo empírico se organiza en las siguientes familias de variables:

1. Variables dependientes  
2. Variables explicativas principales  
3. Variables mediadoras  
4. Variables estructurales  
5. Variables institucionales  
6. Variables de control  
7. Variables dinámicas  

---

## Variables dependientes

### 1. Crecimiento económico
**Constructo teórico:** desempeño macroeconómico dinámico  
**Dimensión:** crecimiento real sostenible  
**Interpretación:** expansión sostenida de la capacidad productiva  
**Indicadores esperados:**
- crecimiento del PIB real per cápita
- crecimiento del ingreso nacional real
- crecimiento del producto real ajustado por población  

---

### 2. Volatilidad macroeconómica
**Constructo teórico:** inestabilidad estructural  
**Dimensión:** variabilidad del desempeño económico  
**Interpretación:** exposición a shocks y ciclos  
**Indicadores esperados:**
- desviación estándar del crecimiento del PIB  
- volatilidad del ingreso real  
- volatilidad del crecimiento per cápita  

---

## Variables explicativas principales

### 1. Diversificación productiva
**Constructo teórico:** complejidad estructural  
**Dimensión:** estructura productiva  
**Interpretación:** capacidad de generar valor agregado diverso  
**Indicadores conceptuales:**
- diversificación exportadora  
- complejidad económica  
- diversificación sectorial  
- estructura productiva no primaria  

---

### 2. Transición energética
**Constructo teórico:** transformación del sistema energético  
**Dimensión:** sostenibilidad estructural  
**Interpretación:** cambio en la matriz energética  
**Indicadores conceptuales:**
- participación de renovables  
- intensidad energética  
- descarbonización  
- electrificación  
- eficiencia energética  

---

## Lógica de construcción empírica

La operacionalización empírica seguirá el siguiente orden:

1. Definición conceptual  
2. Definición teórica  
3. Definición empírica  
4. Selección de indicadores  
5. Evaluación de proxies  
6. Validación de calidad de datos  
7. Armonización temporal  
8. Construcción del panel  
9. Validación estadística  
10. Integración al modelo
