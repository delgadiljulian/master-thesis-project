## Advertencia metodológica

Este documento constituye un **sistema de fichaje bibliográfico**, no una revisión de literatura ni un estado del arte.

Su función es servir como:

- registro de lectura científica  
- sistema de síntesis teórica  
- instrumento de análisis crítico  
- base de construcción del marco teórico  
- herramienta de organización del conocimiento  
- soporte del proceso de investigación bibliográfica  

Actualmente se encuentra en **fase estructural**, sin contenido bibliográfico incorporado.

---

## Propósito del documento

Este archivo define el **formato estándar de análisis de fuentes** que se utilizará en el proyecto.

Toda fuente académica relevante será registrada utilizando esta estructura, garantizando:

- homogeneidad de análisis  
- comparabilidad  
- sistematicidad  
- trazabilidad  
- síntesis estructurada  
- análisis crítico  
- coherencia teórica
